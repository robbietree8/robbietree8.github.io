file operation on server

2019-08-12 约 33 字 预计阅读 1 分钟
目录

服务器文件操作笔记

数据库备份

mysqldump -uroot -p -hlocalhost -P3306 db > /var/lib/mysql/db.sql

数据库恢复

mysql -uroot -p -hlocalhost -P3306 db < db.sql

文件打包

tar -zcf /dst/foo.tar.gz foo

这里有个技巧，尽量按少量的文件夹打包，一来可以缩小打包后的文件大小，二来单次打包速度会较快。 tar操作会比较耗CPU资源，所以在操作的时候，可以把其他耗CPU的服务先暂停掉。

文件传输

scp user@host:/dst/foo.tar.gz .

其他

服务器上操作可以用下tmux神器，帮助你管理session，不至于打包到一半的时候连接断了。 服务器操作，慎用rm。
